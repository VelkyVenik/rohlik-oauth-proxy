Rohlik MCP Proxy

Connect to Rohlik's MCP server from any Claude client (iOS, Android, Web, Desktop).

What is this?

Rohlik's MCP server requires custom authentication headers (rhl-email / rhl-pass) that Claude can't set directly. This proxy bridges that gap — you sign in once with your Rohlik credentials via OAuth and Claude handles the rest.

How to connect

  1. In any Claude client, go to Settings → MCP Servers → Add Remote Server.
  2. Set the Server URL to: {{PUBLIC_URL}}/mcp
  3. Claude will open a sign-in page — enter your Rohlik.cz credentials.
  4. Once authenticated, Claude has access to Rohlik tools. Done!
Important: Your Rohlik email and password are stored on this server, encrypted with AES-256-GCM. They are needed to authenticate with Rohlik's MCP server on each request. Only use this proxy if you trust the server operator. This instance is operated by {{OPERATOR_NAME}} ({{OPERATOR_EMAIL}}). You can revoke access at any time by reconnecting or contacting the operator.

Authentication uses OAuth 2.1 with PKCE. Your credentials are never shared with Claude — only used server-side to forward requests to Rohlik.
